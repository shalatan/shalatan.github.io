## <a name='Principles'></a>Principles

### <a name='Principle2'></a>Principle 2
- Write clear and specific instructions.
    - Tactic 1: Use delimiters to clearly indicate distinct parts of the input like: ```, """, < >, <tag> </tag>, :
    - Tactic 2: Ask for a structured output
    - Tactic 3: Ask the model to check whether conditions are satisfied
    - Tactic 4: "Few-shot" prompting i.e Give successful examples of completing tasks and then ask to perform the task.


### <a name='Principle2'></a>Principle 2
- Give the model time to think.
    - Tactic 1: Specify the steps required to complete a task
    - Tactic 2: Instruct the model to work out its own solution before rushing to a conclusion
